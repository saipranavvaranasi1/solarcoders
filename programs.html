Home/Programs

Three orbits, first line of code to first AI project.

A guided progression that meets students where they are and takes them further than they expected to go. Every crew is small, mentored, project-based — and free.

Orbit 01

Low Orbit — Python Basics

Ages 8–11 · 6 weeks

Variables, loops, and logic taught through drawing, mini-games, and playful puzzles. Students leave able to read and write real code.

  • Draw with code (turtle graphics)
  • Number games & simple logic
  • Your first working program
Orbit 02 — Most popular

Mid Orbit — Build with Python

Ages 11–14 · 8 weeks

Functions, data, and projects. Crews ship real things — a quiz app, a weather bot, a small arcade game — and learn to debug like professionals, meeting NumPy and pandas along the way.

  • Build interactive apps
  • Work with real data (NumPy & pandas)
  • Team project showcase
Orbit 03

Deep Space — Python + AI

Ages 12–14 · 8 weeks

An introduction to machine learning with scikit-learn and TensorFlow. Students train simple models, build an AI chatbot, and explore how to use AI thoughtfully and responsibly.

  • Build an AI chatbot
  • Train a model (scikit-learn / TensorFlow)
  • AI ethics & safety

The toolkit

The real tools, taught in the right order.

We start with Python itself, then add the libraries professionals actually reach for — introduced only when a project needs them.

  1. 01

    Python

    The foundation — the language every later tool is written for.

  2. 02

    NumPy & pandas

    Working with numbers and real datasets, the backbone of data work.

  3. 03

    scikit-learn

    A first, friendly step into machine-learning models.

  1. 04

    TensorFlow

    Training a small neural network to see how modern AI learns.

  2. 05

    Django

    Putting a project on the web so kids can share what they built.

  3. 06

    Good habits

    Debugging, reading errors, and building step by step — for life.
